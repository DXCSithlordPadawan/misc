IBM Rational DOORS 9.6.1 Deployment Documentation Prompt

Goal:
Create a complete documentation with details and configuration package for installing IBM Rational DOORS 9.6.1 on a VMware-hosted Windows Server 2022 environment with Oracle database, LDAP/LDAPS SSO, and mutual TLS, aligned with UK Government Secure by Design (SByD), NCSC guidelines, and CIS Benchmark Level 2.

Requirements:
1. Full Environment Details:
   - VMware VM running Windows Server 2022
   - DOORS installed as standalone server
   - Database: Oracle on separate host
   - Authentication: LDAP integration with Active Directory (username/password over LDAPS)
   - Web Access: DOORS Web Client (DWA)
   - Certificates: Issued by AD Certificate Services, mutual TLS required
   - Default firewall and network configuration
   - VM domain-joined before installation

2. Compliance:
   - UK Gov Secure by Design
   - NCSC Secure Design Principles
   - CIS Benchmark Level 2 for Windows Server 2022

3. Deliverables:
   - architecture.md (Mermaid diagrams for architecture artifacts, full architecture details, Firewall and network details, certificate trust chain)

   - installation.md (Pre-requisite details, Database setup steps, Application installation steps, HTTPS+mTLS implementation, LDAP config steps)
   - YAML configuration files: doors-config.yaml, doors-security.yaml
   - README.md (TOC, usage instructions, embedded diagrams, YAML examples)
   - Comprehensive PDF guide (Executive Summary, Compliance Checklist, all docs, diagrams, YAML examples)
   - PowerPoint (.pptx) for stakeholders

4. Formatting:
   - All documents in Markdown (.md)
   - Diagrams in Mermaid syntax
   - Config files in YAML
   - Bundle everything into a ZIP
   - Include PDF and PPTX versions for distribution
